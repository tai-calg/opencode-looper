## 設計判断

- PrismaKnowledgeRepository の save() は 2 ステップ方式（upsert + $executeRaw）で pgvector の Unsupported 型に対応
- Composition Root で OPENAI_API_KEY の有無により OpenAI/Stub アダプタを切り替え
- 既存 rule パターンと完全に同じ構造で実装（Loader/Actions/Dialog/Table）
- `ai` と `@ai-sdk/openai` パッケージを新規追加
