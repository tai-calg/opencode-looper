# frontend-login-and-sidebar: ログインページ + サイドバーコンポーネント

## タスク ID: frontend-login-and-sidebar
Milestone 1 ステップ D — ログインページ（SKIP_AUTH 開発用ログインボタン付き）とサイドバーナビゲーションコンポーネントの作成。

## 作成ファイル
- `apps/webapp/src/frontend/components/sidebar.tsx` — Client Component、usePathname でアクティブリンク判定
- `apps/webapp/src/app/(public)/login/page.tsx` — Server Component、SKIP_AUTH 環境変数で開発用ログインボタン表示制御

## 設計判断
- 設計ドキュメントの型定義・コードをほぼそのまま採用。Biome の import 順序・フォーマット規約に合わせて微調整。
- サイドバーはナビゲーションのみ（ユーザーセクション・ログアウトはステップ E の auth layout が担当）。
- ログインの Google ボタンは disabled（Milestone 7 で有効化）。

## 次のタスクへの申し送り
- ステップ E（auth-layout-and-pages）が sidebar.tsx を import する。ステップ C の session.loader / logout.action も必要。
- `pnpm verify` 全パス確認済み。
