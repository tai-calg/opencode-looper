## 設計判断
- shadcn/ui init がインタラクティブモードで失敗するため、components.json を手動作成し `npx shadcn add` で直接コンポーネント追加
- globals.css の CSS Variables テーマ（neutral ベース）は shadcn/ui 公式の Tailwind v4 用出力を手動記述
- Prisma schema は空（model なし）で prisma generate を通過させ、ステップ B で本実装を行う前提
- vitest に `passWithNoTests: true` を追加してテスト 0 件でも verify 通過
